📌 Como Funciona

  • Seções organizam o conteúdo em grandes tópicos (ex: Álgebra, Geometria)
  • Capítulos subdividem as seções em aulas específicas
  • Conteúdos dentro de capítulos podem ser: textos, vídeos, PDFs, exercícios
  • Toda essa estrutura é gerenciada pelo banco de dados (Supabase)
  • Nenhum arquivo HTML precisa ser criado para adicionar matérias

Equações do 1º Grau

Uma equação do 1º grau é uma igualdade que contém uma variável (incógnita) com expoente 1. A forma geral é: ax + b = 0

Exemplo:
2x + 4 = 10
2x = 6
x = 3
